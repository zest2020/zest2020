# 😀 简介
- Sailor编程工作室是一个由在校大学生组成的编程团队，团队主要成员来自于河南牧业经济学院北林校区信息工程学院（软件学院）和能源与智能工程学院的同学
- 我们很菜，但我们很热爱---热爱生活、热爱编程。
- 这里是一个有爱的平台，欢迎河南牧业经济学院北林校区对编程有兴趣的同学加入我们
